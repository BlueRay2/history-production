# Hook Deployment Matrix

**Источник:** consensus run 2026-04-11.
**Назначение:** выбор правильного типа хука под (a) персону, (b) тип города, (c) доступные нарративные материалы.

## Восемь типов хуков (базовая таксономия)

1. **Paradox — contradiction** («Should not have existed / lasted / worked»)
2. **Paradox — inversion** («Everyone thinks X, actually Y»)
3. **Paradox — textbook wrong** («The textbook version is incomplete»)
4. **Paradox — "and yet"** («For all its glory, and yet...»)
5. **Stakes-forward — if X failed** («If this city had fallen, Y would never exist»)
6. **Stakes-forward — one choice decided** («One decision set the entire course»)
7. **Scene-drop — sensory** (immediate sensory immersion — smell/sound/street life)
8. **Scene-drop — primary quote** (open with a historical quote or letter reading)

К ним consensus run добавляет три **мета-хука** из Codex «opening promise» matrix, которые можно комбинировать с восемью:

- **Mechanism** («became powerful because of ONE thing nobody could replace»)
- **Contest** («for a thousand years, empires fought over this»)
- **Transformation** («three different cities wearing the same name»)
- **Hidden engine** («the story isn't the kings — it's the water, the walls, the taxes»)

## Выбор хука: decision tree

**Шаг 1: насколько известен город?**
- **Очень известен** (Paris, Rome, London, Tokyo) → хуки 2, 3 (мягкое), 4, mechanism
- **Умеренно известен** (Istanbul, Amsterdam, Kyoto) → хуки 1, 5, 6, mechanism, contest
- **Мало известен** (Quanzhou, Mari, Samarkand) → хуки 1, 2, 7, transformation, hidden engine

**Шаг 2: есть ли сильный primary source?**
- **Есть** (letter, chronicle, diary) → хук 8 (primary quote) как opening device
- **Нет** → выбрать из остальных

**Шаг 3: тип narrative?**
- **Rise + Fall (трагическая арка)** → хук 4 (and yet) + Kings and Generals stake-framing
- **Continuous reinvention** → transformation (meta)
- **Strategic importance** → contest (meta) + Kings and Generals stake-framing
- **Hidden mechanism story** → hidden engine (meta) + Real Engineering "textbook vs reality"
- **Cultural crossroads** → scene-drop sensory (7) или primary quote (8)

**Шаг 4: соответствие персонам**
- Если выбранный хук 🔴 для Persona B (которая 41.7%) — пересмотреть или смягчить.
- Если конфликт — **compromise hook** = 6 (one choice decided) или mechanism.

## Anti-patterns — что никогда не делать в хуках

1. **"Hi guys welcome back to the channel"** — мгновенный kill обеих персон.
2. **"You won't believe what happened"** — кликбейт-язык, отталкивает Persona B, Persona A считает low-quality.
3. **"Like and subscribe"** в первые 30 секунд — 99.5% новых viewer'ов, они не знают вас.
4. **Self-referential jokes** («as I mentioned in the last video») — 99.5% новых.
5. **ALL CAPS в первой строке** — клик-бейт сигнал.
6. **Academic warm-up** («In this essay I will...») — отталкивает Persona A.
7. **«Let me take you on a journey...»** — tour-guide cliché, отталкивает обе персоны.
8. **Generation-specific slang** (от «bro» до «literally») — Persona B блокирует.
9. **Сарказм / ирония в первых 30 секундах** — слишком рано, риск misread.
10. **Долгий чрезмерно формальный интро** («In the year 1271 during the reign of...») — Persona A уходит.

## Примеры: quanzhou — конкретные опции

Для первого видео под consensus findings (Quanzhou, 16-22 мин long-form, малоизвестный город):

### Option A (рекомендация — Claude R1)
**Hook type:** Stakes — one choice decided + Primary quote immersion
**Text:**
> «In 1271, a Persian merchant wrote a letter from a Chinese port. That letter tells us more about the medieval world than any European source. The city he wrote from is where half the world met — and almost no one today remembers its name.»

**Почему работает:**
- Stakes - one choice decided 🟢🟢 обе персоны.
- Primary source immersion 🟡🟢.
- Тесис-first по паттерну Historia Civilis.
- Paradox built-in («half the world», «almost no one remembers»).
- Нет clickbait-языка.

### Option B (backup — Scene-drop sensory)
**Text:**
> «Imagine walking through a city in 1271 where Arabic, Tamil, Persian, Chinese, and Latin were all spoken on the same street. It wasn't Córdoba. It wasn't Venice. It was Quanzhou — and within two centuries, it would disappear from world history entirely.»

**Почему работает:**
- Scene-drop sensory 🟢🟡.
- Transformation (meta) 🟢🟡.
- Встроенная inversion («не Córdoba, не Venice»).
- Сильнее для Persona A, приемлемо для B.

### Option C (experimental — Hidden engine)
**Text:**
> «The largest port in the medieval world wasn't in Europe. It wasn't in Arabia. It was a Chinese city you've probably never heard of — and the reason it vanished isn't what you think.»

**Почему работает:**
- Hidden engine (meta) 🟢🟢.
- Textbook wrong (softened) 🟡.
- Сильный cold open для 99.5% new viewers.

**Рекомендация от consensus: ship Option A как первичная; test Option B и C как title/hook variants через YouTube Test & Compare.**

## Hook × runtime map

Для **Anchored Cascade** структуры (см. round-3.md consensus point 2):

| Runtime zone | Назначение | Hook type relevance |
|---|---|---|
| 0:00-0:30 | Opening hook (один из 8/11 типов) | Выбор из матрицы выше |
| 0:30-2:00 | Hook expansion, build trust | Не новый хук — доставка payoff'а из opening |
| 2:00-4:00 | Trust-building micro-payoffs | Не хук |
| 4:00-5:00 | **SECOND THESIS** / Act 1 cliffhanger | Новый **mini-hook** — второе обещание |
| 5:00-8:00 | Payoff против second thesis | Не хук |
| 8:00-12:00 | Reversal cluster | **Третий mini-hook** возможен — «here's the real turn» |
| 12:00-end | Externalization | Callback на opening image |

**Важно:** «хук» — не только на старте. Второй тесис на 4-5 минуте — тоже мини-хук, и его тип должен быть **контрастным** к opening. Если opening был Paradox, second thesis может быть Mechanism или Transformation. Это создаёт schema disruption необходимый для удержания внимания через Act 1 audit.

## Быстрая памятка сценаристу

1. **Первое предложение** — thesis, не intro.
2. **Первый абзац (0-20с)** — устанавливает образ + напряжение + обещание.
3. **Второй абзац (20-45с)** — hidden engine или mechanism.
4. **Третий абзац (45-60с)** — roadmap как конфликт, не хронология.
5. **Второй тесис** — планируется на минуту 4-5, должен быть **крупнее** опенинга.
6. **Reversal cluster** — планируется на 8-12, 2-3 трансформации близко друг к другу.
7. **Финал** — callback на opening image с новым смыслом.

Если сценарий это делает — хорошая заготовка для Cascade. Если нет — переписать.
