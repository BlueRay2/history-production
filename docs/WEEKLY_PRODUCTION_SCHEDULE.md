# Weekly Production Schedule — history-production (shorts)

**Основной ритм:** 3 шортса в неделю, публикация — **понедельник**.

> Long-form трек — в ветке `main`, файл `docs/WEEKLY_PRODUCTION_SCHEDULE.md`.

Якорь — понедельник (публикация шортсов).

---

## Shorts pipeline (Артём)

Цель: **3 шортса в неделю**, публикация — **понедельник**.

| День | Кто | Что |
|---|---|---|
| **Вт–Пт** | Артём | Производство 3 шортсов (скрипт → футажи → монтаж) |
| **Пт (вечер)** | Артём | Выкладывает черновики в Production forum для ревью |
| **Сб** | Ярослав + команда | Правки по шортсам (на общей встрече продакшена) |
| **Сб–Вс** | Артём | Реализация правок, финальная версия |
| **Пн** | Артём | **Публикация** шортсов |
| **Пн (после публикации)** | Ярослав | Финальная проверка опубликованных шортсов. Если всплывают косяки — Артём делает hot-fix |

---

## Summary (быстрый взгляд)

```
Вт  → старт производства новой тройки шортсов
Ср  → работа над шортсами (скрипты, футажи)
Чт  → работа над шортсами (монтаж)
Пт  → вечер: черновики в Production forum → ревью
Сб  → правки на встрече продакшена
Вс  → реализация правок
Пн  → публикация ← якорь + финальный контроль качества
```

---

## Роли в ревью шортсов

- **Артём** — производство от и до, hot-fix после публикации при необходимости
- **Ярослав** — финальный review перед публикацией и после неё
- **Lead (Claude)** — прогон через тот же стек, что и long-form, если Артём просит: сценарий, идеи, proofreading
- **Codex** — video review через GPT-5.4 Vision (если нужен second pair of eyes на динамику/монтаж)
- **Gemini** — research / hook ideas для виральности

---

## Взаимодействие с long-form треком

- Во время **чт** Артём может параллельно помогать Диме с генерацией футажей для long-form (если его шортсы на этот момент в стадии, где это не мешает)
- Шортсы НЕ обязаны быть привязаны к теме long-form недели — Артём свободен в выборе исторических сюжетов
- Если тема шортса перекликается с выходящим вторничным видео — это плюс для cross-promo, но не требование
