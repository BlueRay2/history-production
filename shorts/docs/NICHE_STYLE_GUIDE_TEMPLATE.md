# Niche Style Guide — Шаблон адаптации под канал (YouTube Shorts)

> **Формат:** YouTube Shorts — вертикальное видео ≤60 секунд, без голосового сопровождения, без текстового нарратива.
> Только визуальный сторителлинг + музыка/звук. Генерация видео — исключительно Kling AI.
>
> **Как использовать:** Заполни все секции ниже, заменив плейсхолдеры `{{...}}` на данные своего канала.
> Сохрани готовый файл как `docs/ref/NICHE_STYLE_GUIDE.md` в рабочей директории проекта скрипта.
> Этот документ читают **все агенты** команды — он определяет творческие рамки всего продакшена.

## 6. Структура и формат Shorts

### Длительность
**Целевая:** {{SHORTS_DURATION — например: 30-50 секунд (оптимум для алгоритма)}}
**Абсолютный максимум:** 60 секунд

### Стандартная структура
{{SHORTS_STRUCTURE — описать структуру ролика. Например:
1. **Хук (0-3 сек)** — визуальный удар: неожиданный кадр, контраст, движение. Зритель должен остановить скролл.
2. **Развитие (3-25 сек)** — раскрытие визуальной истории. Нарастание масштаба, детализации или напряжения.
3. **Кульминация (25-40 сек)** — пиковый момент: самый эффектный кадр, трансформация, развязка.
4. **Луп-бэк (последние 3-5 сек)** — визуальный переход, который естественно возвращает к первому кадру. Зритель пересматривает — алгоритм вознаграждает.}}

### Частота выпуска
{{UPLOAD_FREQUENCY — например: 4-7 Shorts в неделю, ежедневно или через день}}

### Серийность
{{SERIES_STRATEGY — например:
- Тематические серии по 5-10 роликов (один мир/концепт)
- Узнаваемый визуальный "почерк" серии (единая палитра, мотив)
- Standalone ролики для вирального охвата между сериями}}
