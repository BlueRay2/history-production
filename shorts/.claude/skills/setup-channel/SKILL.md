# Setup Channel — Настройка профиля Shorts-канала

## Назначение

Первичная настройка форка ИЛИ обновление существующего профиля канала.
Заполняет секции 1-8 и 10 шаблона NICHE_STYLE_GUIDE — всё, кроме секции 9 (тема конкретного видео).

Профиль адаптирован под YouTube Shorts: вертикальный формат 9:16, длительность ≤60 секунд, без голосовой озвучки и текстового нарратива, генерация видео только через Kling AI.

## Алгоритм

### 1. Подготовка

- READ `docs/NICHE_STYLE_GUIDE_TEMPLATE.md` — шаблон со всеми плейсхолдерами.
- READ `docs/ref/NICHE_STYLE_GUIDE.md` если существует — для обновления (не перезаписывать заполненные секции без необходимости).
- Если файл существует и секции 1-8, 10 заполнены (нет `{{...}}`), сообщить пользователю и спросить, хочет ли он обновить профиль.

### 2. Интерактивный опрос

Провести пользователя по **6 группам вопросов** последовательно через AskUserQuestion.

**Для каждой группы** начать с AskUserQuestion, предлагающего **3 варианта**:

| Вариант | Описание |
|---------|----------|
| **Пропустить** | Оставить поля группы пустыми (плейсхолдеры `{{...}}`) для заполнения позже |
| **Автозаполнение** | Claude генерирует ответы из доступного контекста сессии (предыдущие ответы, загруженные документы, контекст беседы). Показать результат пользователю для подтверждения/правки |
| **Свободная форма** | Задать вопросы группы, пользователь отвечает на часть или все. Поля без ответа автозаполняются из контекста других ответов и сессии |

#### Группа 1 — Канал и ниша (секция 1)
Поля:
- Название канала
- URL канала
- Ниша / контентная вертикаль
- Поднишевая специализация
- Язык контента

#### Группа 2 — Аудитория (секция 2)
Поля:
- **Демография:** возраст, пол (соотношение), география, образование, доход
- **Психография:** ценности, страхи, стремления, как потребляют контент, уровень подготовки
- **Болевые точки:** 3-5 пунктов, что беспокоит аудиторию в контексте ниши

#### Группа 3 — Бренд и визуальный стиль (секции 3-4)
Поля:
- Бренд-персональность для визуального контента: эмоциональность (1-10), юмор (1-10 + тип визуальных гэгов)
- Стиль текстовых оверлеев (если используются): шрифт, размер, расположение, анимация
- Конвенции хештегов: основные хештеги канала, правила подбора хештегов к теме
- 2-3 примера визуальных приёмов в стиле канала
- Запрещённые визуальные элементы и приёмы

#### Группа 4 — Визуал и структура (секции 5-6)
Поля:
- Цветовая палитра (основные цвета, настроение грейдинга)
- Визуальный стиль (описание + референсы)
- Визуальные запреты
- Длительность видео: ≤60 секунд (указать предпочтительный диапазон, напр. 30-45 сек, 50-60 сек)
- Структура Shorts: Hook (0-5 сек) → Развитие → Кульминация → Зацикливание (loop-back на начало)
- Частота выпуска (учитывая объёмы Shorts, напр. ежедневно, 3-5 в неделю)
- Предпочтения вертикальной композиции 9:16: расположение ключевых объектов, зоны внимания, работа с верхней/нижней частью кадра (UI YouTube перекрывает)

#### Группа 5 — Бизнес (секции 7-8)
Поля:
- Модель монетизации
- Формат рекламных интеграций (если есть)
- Прямые конкуренты (3-5 каналов с URL, сильные/слабые стороны)
- Стилистические референсы
- Чем канал отличается (2-3 предложения)

#### Группа 6 — Технический стек (секция 10)
Поля:
- Генерация видео: Kling AI — версия модели (напр. Kling 2.1, Kling 1.6), уровень подписки (Free / Standard / Pro / Pro+)
- Источник музыки: платформа/библиотека (напр. Epidemic Sound, YouTube Audio Library, Suno AI), жанровые предпочтения
- Монтаж: программа, формат экспорта (вертикальный 1080x1920)

### 3. Секция 9 — НЕ заполняется

Секция 9 (Тема текущего видео) заполняется отдельно через `/setup-video` перед каждым новым скриптом. Оставить плейсхолдеры `{{...}}` в секции 9 без изменений.

### 4. Генерация файла

- `mkdir -p docs/ref`
- Записать `docs/ref/NICHE_STYLE_GUIDE.md` на основе шаблона, заменив плейсхолдеры в секциях 1-8 и 10 ответами пользователя.
- Секцию 9 скопировать из шаблона без изменений (с плейсхолдерами).
- Для групп с выбором "Пропустить" — оставить `{{...}}` как есть.

### 5. Валидация

- Проверить: в секциях, которые не были пропущены, нет оставшихся `{{...}}`.
- Если есть незаполненные поля в непропущенных группах — запросить у пользователя.

### 6. Пост-setup

Сообщить пользователю:
- Профиль канала сохранён в `docs/ref/NICHE_STYLE_GUIDE.md`
- Если были пропущенные группы — перечислить какие секции остались с плейсхолдерами
- Для начала работы над видео выполни `/setup-video`
