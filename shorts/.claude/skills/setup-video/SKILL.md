# Setup Video — Настройка темы Shorts-видео

## Назначение

Заполнить или обновить секцию 9 (Тема текущего видео) в `docs/ref/NICHE_STYLE_GUIDE.md`.
Вызывается перед началом работы над каждым новым Short.

Формат: вертикальное видео 9:16, до 60 секунд, без голосового сопровождения, без текстовой нарации — только визуальный ряд + музыка/звук. Генерация исключительно через Kling AI.

## Предусловие

Файл `docs/ref/NICHE_STYLE_GUIDE.md` должен существовать с заполненными секциями 1-8 и 10.
Если файл не найден или секции 1-8/10 содержат плейсхолдеры `{{...}}` — направить пользователя на `/setup-channel`.

## Алгоритм

### 1. Проверка

- READ `docs/ref/NICHE_STYLE_GUIDE.md`
- Убедиться, что файл существует.
- Проверить секции 1-8 и 10: не должно быть `{{...}}` плейсхолдеров (кроме секции 9).
- Если предусловие не выполнено — сообщить и предложить `/setup-channel`.

### 2. Интерактивный опрос

Начать с AskUserQuestion, предлагающего **3 варианта**:

| Вариант | Описание |
|---------|----------|
| **Пропустить** | Оставить плейсхолдеры `{{...}}` в секции 9 для заполнения позже |
| **Автозаполнение** | Claude генерирует ответы из контекста сессии (если пользователь уже обсуждал тему видео в беседе). Показать результат для подтверждения/правки |
| **Свободная форма** | Задать вопросы по полям, пользователь отвечает на часть или все. Поля без ответа автозаполняются из контекста ответов и сессии |

Поля секции 9:
- **Рабочее название видео** — {{VIDEO_TITLE}}
- **Визуальная концепция** (1-2 предложения: что зритель увидит, какая визуальная история/трансформация) — {{VIDEO_TOPIC}}
- **Визуальная история** (какая трансформация, метаморфоза или визуальное путешествие разворачивается за 60 секунд) — {{VISUAL_CONCEPT}}
- **Референс-пример** (ссылка на существующее видео/Short + описание: что именно нравится — переходы, визуальный стиль, ритм, настроение. Помогает точно передать задуманный визуал) — {{VIDEO_REFERENCE_EXAMPLE}}
- **Целевое настроение** — {{TARGET_MOOD}}
- **Настроение музыки/звука** (какая музыка или звуковое сопровождение подходит: жанр, темп, энергия) — {{MUSIC_MOOD}}
- **Стратегия зацикливания** (как конец видео визуально соединяется с началом для loop-эффекта; или "без зацикливания") — {{LOOP_STRATEGY}}
- **Обязательные визуальные элементы** (3-7 пунктов: объекты, сцены, переходы, которые точно должны быть) — {{MUST_INCLUDE}}
- **Чего НЕ должно быть** (что исключить) — {{MUST_EXCLUDE}}

### 3. Обновление файла

- Заменить ТОЛЬКО секцию 9 в `docs/ref/NICHE_STYLE_GUIDE.md`.
- Не трогать секции 1-8 и 10.
- Использовать Edit tool для точечной замены секции 9.
- При выборе "Пропустить" — оставить `{{...}}` как есть.

### 4. Валидация

- Если выбрана "Свободная форма" или "Автозаполнение": проверить, что в секции 9 нет оставшихся `{{...}}`.
- Если после автозаполнения остались `{{...}}` — перечислить пользователю какие поля не удалось заполнить из контекста и запросить ввод.

### 5. Пост-setup

Сообщить пользователю:
- Тема Shorts-видео заполнена в `docs/ref/NICHE_STYLE_GUIDE.md`, секция 9.
- Можно запускать `/shorts-storyboard` для создания визуального сценария.
