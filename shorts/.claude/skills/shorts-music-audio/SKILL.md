# Музыка и звуковой дизайн для YouTube Shorts

## 1. Роль музыки в Shorts

Без войсовера музыка — это и есть нарративная структура. Она задаёт:
- Эмоциональный тон
- Темп и ритм (монтажные склейки синхронизируются с битами)
- Нарастание и кульминацию
- Ощущение петли (бесшовный аудио-луп = бесшовный видео-луп)

## 2. Источники музыки

### YouTube Audio Library (бесплатно)
- Встроенная библиотека, безопасна для монетизации
- Ограниченный выбор, но есть фильтры по настроению, жанру, темпу

### Epidemic Sound / Artlist (подписка)
- Профессиональное качество
- Stems (разделённые дорожки) для точной синхронизации
- Epidemic ~$15/mo, Artlist ~$10/mo

### Трендовые звуки YouTube Shorts
- YouTube Shorts имеет систему трендовых звуков (как TikTok)
- Использование трендового звука увеличивает охват
- Мониторить через YouTube Studio → Shorts → Trending sounds

### Kling 2.6 Native Audio
- Kling 2.6 генерирует аудио вместе с видео
- Стоимость: 50-200 кредитов за клип (vs 10-70 без аудио)
- Хорошо для: ambient sounds, SFX, atmospheric audio
- Плохо для: музыкальный ритм, точная синхронизация
- Используй для отдельных клипов, финальную музыку накладывай в монтаже

## 3. Музыкальная структура для Shorts

### Формула 60-секундного Short:
```
[0-5s]   Intro/hook — сильный бит или distinctive звук
[5-15s]  Build-up — нарастание, добавление элементов
[15-40s] Main groove — основной ритм, пик энергии
[40-55s] Climax/drop — кульминация
[55-60s] Outro/loop — плавный переход к началу
```

### BPM и монтаж
- Slow/atmospheric: 70-90 BPM → 1 cut per 2-4 beats
- Medium/narrative: 90-120 BPM → 1 cut per 1-2 beats
- Fast/energetic: 120-160 BPM → 1 cut per beat or faster

### Синхронизация
- Каждая смена клипа = сильная доля (beat)
- Резкие визуальные изменения = музыкальные акценты (удар, дроп)
- Плавные переходы = переходные элементы музыки (свеллы, ризеры)

## 4. Аудио-промпты для Kling 2.6

При использовании Kling 2.6 для генерации аудио добавь описание аудио в промпт:

### Формат
Добавляй в конец любого Kling-промпта: аудио-контекст описывает, что должно быть слышно.

### Примеры аудио-описаний:
- "Ambient city sounds, distant traffic, rain on pavement, soft reverb"
- "Mechanical whirring, electronic hum, subtle sci-fi atmosphere"
- "Ocean waves crashing, seagulls in distance, wind"
- "Footsteps on gravel, rustling leaves, bird chirping"

### Когда использовать Kling 2.6 audio vs внешнюю музыку:
| Сценарий | Рекомендация |
|----------|-------------|
| Атмосферные Shorts (nature, ASMR) | Kling 2.6 audio — ambient sounds |
| Ритмичные Shorts (montage, transformation) | Внешняя музыка — нужен чёткий ритм |
| Satisfying/process Shorts | Kling 2.6 для SFX + внешняя музыка |
| Трендовый звук | Только внешняя музыка (трендовый трек) |

## 5. Бюджет: аудио от Kling 2.6

Генерация с аудио стоит значительно дороже:
- T2V 5s Standard без аудио: 10 кредитов
- T2V 5s Standard с аудио: ~50 кредитов (5×)
- T2V 10s Professional без аудио: 70 кредитов
- T2V 10s Professional с аудио: ~200 кредитов (3×)

Рекомендация: генерировать видео БЕЗ аудио, накладывать музыку в монтаже. Использовать Kling 2.6 audio только для клипов, где нужен синхронный ambient sound.
